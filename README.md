### Shunting yard calculator (calculator/calc.cpp)
A simple calculator implementing the Shunting Yard algorithm to parse and evaluate arithmetic expressions given as strings. Supports addition, subtraction, multiplication, division, and parentheses.

#### Usage examples
```
Expression: (2 + 2) * 2
(2 + 2) * 2 = 8

Expression: 2 + 2 * 2
2 + 2 * 2 = 6
```
#### Manual Compilation
```bash
clang++ calculator/calc.cpp -o calc/calculator
./calc/calculator
```
